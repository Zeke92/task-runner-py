Task Runner Py - SSD Space Monitor

Un script ligero en Python diseñado para monitorear el espacio disponible en tu SSD o disco duro y lanzar alertas del sistema cuando el espacio cae por debajo de un límite configurado (por ejemplo, 150 GB).

---
 Características

* **Monitoreo automático:** Revisa el estado del disco en intervalos regulares (por defecto, cada 1 hora).
* **Notificaciones del sistema:** Muestra alertas nativas en pantalla cuando el espacio es crítico.
* **Sin alertas repetitivas:** Evita hacer *spam* de notificaciones; solo vuelve a alertar si el espacio se recupera y vuelve a caer.
* **Fácil de configurar:** Modifica la ruta del disco, el límite de almacenamiento o el tiempo entre revisiones en un par de líneas.
